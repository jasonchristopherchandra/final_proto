About Chat Translator




ChatTranslator is a new web application focusing on translating YouTube live stream chat box messages while also providing users with a built in translator to translate any messages that they wish to send into the live stream chatbox. Currently. we support English to Indonesian chatbox message translation, and Indonesian to English user message translation.




Our application currently supports English to Indonesian chatbox message translation, and Indonesian to English user message translation. However, we will continue develpoment for this web application to include more languages and live streaming platforms supported.
